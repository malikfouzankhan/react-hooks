:: Why React?
While everything we do in React can technically be done in plain HTML, CSS and JS. React allows:
    - Better code reuse
    - Large-scale performance in production
    - Clean separation of concerns via components
    - Easier maintenance and testing

:: Folder Structure
Industry Standard Folder Layout:
    src/
     |-- components/    # Reusable UI components
     |-- pages/         # Page-specific components
     |-- utils/         # Constants, helpers and mock data

:: Importing files in React

You can import like this:
    import Header from "./components/Header.jsx     // you can skip the extension

    Note: The .jsx extension is optional while importing
